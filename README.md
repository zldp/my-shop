## 项目说明
此项目是一个学习项目，用到一个Maven的一个模块化开发，模块不是很细
* my-shop-commons （公共模块）
* my-shop-dependencies （Maven依赖关系）
* my-shop-domain （实体类）
* my-shop-web-admin （controller service dao都在这里）
* my-shop-web-api （我特么忘记了）
* my-shop-web-ui （前端界面）
